解释

  1. 使用指定的数据库：

    确保我们正在使用 SPJ_MNG 数据库。

  2. 更改语句结束符：

    将语句结束符从默认的分号 ; 改为 //，以避免在存储过程中使用分号时引起的问题。

  3. 创建存储过程：

    • 声明变量：声明必要的变量。
    • 声明游标：声明游标 cur，用于查询供应该工程号 job_no 零件的所有供应商的名称。
    • 声明继续处理程序：当游标到达末尾时，设置 done 为 TRUE。
    • 创建临时表：创建一个临时表 temp_suppliers，用于存储唯一的供应商名称。
    • 开启游标：打开游标。
    • 循环处理游标：使用 LOOP 循环从游标中获取数据，并将供应商名称插入到临时表中，自动去重。
    • 关闭游标：关闭游标。
    • 从临时表中读取供应商名称并拼接：使用 GROUP_CONCAT 函数将临时表中的供应商名称拼接成一个字符串，并用逗号和空格分隔。
    • 删除临时表：删除临时表，释放资源。
